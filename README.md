# EasyTunnel — Landing Page

Marketing / SEO landing page for **EasyTunnel**, an iPhone-first SSH client with local
port forwarding, a terminal, SFTP, and a code editor.

- App source: https://github.com/noelmom/easytunnel
- Status: App Store — coming soon

## Hosting

Static site (plain HTML/CSS). Serve with GitHub Pages (Settings → Pages → Deploy from `main` / root)
or any static host.

> **Before going live:** update the canonical URL, `og:url`, `sitemap.xml`, and `robots.txt`
> to your real domain, and add a `CNAME` file if you use a custom domain (e.g. `easytunnel.app`).
> Replace the "Coming soon" App Store badge in `index.html` with the real App Store link when the app ships.

## Files
- `index.html` — the page
- `styles.css` — styling (brand palette)
- `assets/icon.png` — app icon
- `assets/og-image.png` — 1200×630 social/preview image
