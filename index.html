EasyTunnel

Simple tunnels. Powerful tools.

An iPhone-first SSH client. Forward local ports and browse them right in the app, run a full terminal, manage files over SFTP, and edit code — all in one clean interface.

Coming soon to the App Store

iPhone · iOS 17+ · Private & on-device — no accounts, nothing phones home.

Everything you need for SSH on the go

⌨️

Full SSH terminal

A real interactive terminal that adapts to light and dark mode, with an on-screen key bar for esc, ctrl, and arrows.

🔀

Local port forwarding

Set up ssh -L forwards and open several at once. Reach services behind your SSH host as if they were local.

🌐

Built-in web view

Browse a forwarded web app at localhost without leaving EasyTunnel — camera and microphone work too.

📁

SFTP file browser

Browse, upload, download, rename, create folders, and delete remote files. Preview text and images with pinch-to-zoom.

📝

Code editor

Edit remote files with syntax highlighting and a Markdown preview, then save straight back over SFTP.

🔐

Flexible authentication

Password, private key (ed25519 & ECDSA), or keyboard-interactive for 2FA and one-time codes — with a reusable key library.

🛡️

Host-key pinning

Trust-on-first-use pins each server's key and warns you if it ever changes — protection against man-in-the-middle attacks.

🔄

Sessions that survive

Quick app-switches stay connected, sessions auto-reconnect when you return, and idle sessions can disconnect on their own.

⚡

Quick Connect

Fire off an ad-hoc session without saving anything, or keep tunnels handy on a dashboard that shows what's live at a glance.

Built for people who live in the terminal

EasyTunnel keeps SSH, port forwarding, and file transfer simple on iPhone — no clutter, no accounts, no data leaving your device.
